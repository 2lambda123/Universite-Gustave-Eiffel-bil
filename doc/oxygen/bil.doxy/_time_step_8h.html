◆ TimeStep_ComputeTimeStep()

double() TimeStep_ComputeTimeStep ( TimeStep_t *  timestep,
Solution_t *  soln,
double  t1,
double  t2 
)

Return the current time increment dt so that the current time is t = tn + dt. The two times tn and t should lie within the range between t1 and t2. Also set the location of the time step dt within the range between t1 and t2 as being either at the beginning, in between or at the end of the range [t1:t2].
