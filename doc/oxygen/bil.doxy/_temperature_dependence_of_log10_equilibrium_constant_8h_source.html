25 /*
26  * The Gibbs free energy change of reaction,
27  *
28  * DrG = SUM_i (nu_i * mu_i)
29  *
30  * is equal to 0 at equilibrium: DrG = 0. The ln(K) is linked to the Gibbs free
31  * energy change of the reaction in standard state, DrG0 = SUM_i (nu_i * mu0_i):
32  *
33  * DrG0 = - RT * ln(K)
34  *
35  * which itself is linked to the formation Gibbs free energy: DrG = SUM_i nu_i DfG(i).
36  *
37  * We compute DrG0 through the followings steps
38  *
39  * DrG(T) = DrH(T) - T * DrS(T)
40  * DrH(T) = DrH(T0) + Int_T0^T DrCp(T) dT
41  * DrS(T) = DrS(T0) + Int_T0^T DrCp(T)/T dT
42  *
43  * and so
44  *
45  * DrG(T) = DrG(T0) - (T - T0) * DrS(T0) + (Int_T0^T DrCp(T) dT) - T * (Int_T0^T DrCp(T)/T dT)
46  *
47  * ie.
48  *
49  * ln(K) = ln(K0) - DrH(T0) / R * (1/T - 1/T0) - (1/RT) * (Int_T0^T DrCp(T) dT) + (1/R) * (Int_T0^T DrCp(T)/T dT)
50  *
51  *
52  * Three-term temperature extrapolation
53  * ------------------------------------
54  * This corresponds with a constant heat capacity of reaction DrCp(T) = DrCp. Then
55  *
56  * ln(K) = ln(K0) - (DrH(T0)/R) * (1/T - 1/T0) + (DrCp/R) * ( T0/T - 1 + ln(T/T0) )
57  *
58  *
59  * So with the approximation: log(K) = A1 + A2 * T + A3 / T + A4 * log(T) + A5 / (T*T)
60  *
61  * A1 = + (1/(ln(10) * R)) * ( (DrS(T0)) - (DrCp) * ( 1 + ln(T0) )
62  * A2 = 0
63  * A3 = - (1/(ln(10) * R)) * ( (DrH(T0)) - (DrCp) * T0 )
64  * A4 = + (1/(R)) * (DrCp)
65  * A5 = 0
66  */
